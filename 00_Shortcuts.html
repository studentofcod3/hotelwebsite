Great shortcuts

-Hold control+enter to skip to the next line without moving any text.

-Hold shift+alt then down to copy a line of text down.

-Hold control+left/right to skip across words. -Hold alt+up/down to move a line up or down.

-! then tab gives us the boilerplate for html5 folders

-Hold control+d whilst a certain piece of text is highlighted. This will highlight the next text which matches. Can then edit them at the same time.

-Hold shift+tab whilst a section of text is highlighted to bring it backwards

-Hold control+/ to comment in/out pieces of text (works for both html and css although appears differently for each

-When typing elements you can hit *n which gives that element repeated n times

-SHORTCUT FOR TYPE: ie: if you use input:email you can put the type in after the colons; email type in this case

-SHORTCUT FOR IDs: when typing an element add # followed by id of choice. [NOTE: for divs you can simply put #ID and it will make a div with the id of ID]
-For Class the same but with a .
-These two can even be joined

-HOLD shift+back to highlight text
